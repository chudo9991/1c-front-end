Ниже реализован "список дел". Можно написать название дела и нажать enter. Оно добавиться в список. При клике на него дело удалиться из списка.

Список дел

  1. {{ item }}

Мы реализовали это с помощью вот такого кода в HTML:

            
            <div id="app">
                <h3>Список дел</h3>
                <input type="text" v-model="thing" v-on:keydown="addItem($event)">
                <ol>
                    <li v-for="(item,index) of list"
                        v-on:click="removeItem(index)">{{ item }}</li>
                </ol>
            </div>

И вот такого кода в файле 64.js:

            
            let vm = new Vue ({
                el: "#app",
                data: {
                    list : ["Мой руки!", "Делай уроки!", "Ложись спать!", "Автор: фреккен Бок из Карлссона"],
                    thing : ""
                    },
                methods : {
                    addItem: function (event) {
                        if (event.which==13) { 
                            this.list.push(this.thing); 
                            this.thing="";
                        }
                    },
                    removeItem : function (index) { 
                        this.list.splice(index,1);  
                    }
            
                }
            });

В словаре data мы задаем заранее подготовленный список задач list и переменную thing в которую будем вносить данные от пользователя.

В словаре methods мы определяем работу двух методов: addItem и removeItem. Причем в addItem мы передаем словарь event. А в removeItem номер элемента в списке, на котором кликнул пользователь.

$event - стандартный элемент DOM-модели, передается как словарь.
