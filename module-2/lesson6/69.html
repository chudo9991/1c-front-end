Директивы v-if и v-show управляют видимостью элемента в зависимости от значения выражения.

Если выражение ложно, от v-if удаляет элемент целиком, а v-show лишь скрывает его при помощи display: none.

Скрываем элементы:

Now You See Me!

Код в HTML:

        <div id="app">
            <h3>Скрываем элементы:</h3>
            <input type="checkbox" v-model="visible">
            <h1 v-show="visible">Now You See Me!</h1>
        </div>

Код в 610.js:

        let vm = new Vue ({
            el: "#app",
            data: {
                visible: true 
            }
        });
