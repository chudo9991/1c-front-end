7.1.4 Создание модулей

Все переменные и функции модуля по умолчанию являются локальными. Для создания экспортируемого объекта возвращаемого функцией require, используют переменную module.exports.

Приведем пример.

        Листинг 7.4 dependency.js

        function f(x) {
            ...
        }
        module.exports = {
            act: f
        };
        
        Листинг 7.5. main.js 
        
        let dependency=require("/.dependency.js");

        ... 
        dependency.act(42);
