7.1.2 Автоматическая сборка

Чтобы не вызывать вручную команду компиляции каждый раз при изменении main.js, можно включить автоматическую сборку в момент сохранения при помощй пакета Watchify. Устанавливаем его.

Это делаем в Windows Power Shell и Node.js!

  1. Запустить Windows PowerShell
  2. Набрать 'npm install --global watchify'.

Проверм, как это работает. Заменим текст в команде alert в main.js.

        Листинг 7.1 index.html 

        <!DOCTYPE html>
        <html>
        <head>
            <meta charset="uft-8">
            <title>Browserify</title>
        </head>
        <body>
            <button type="button">Alert!</button>
            <script src="bundle.js"></script>
        </body>
        </html>
    
        Листинг 7.2 main.js

        let $=require("jquery")
        $("button").on("click", function() {
            alert("Hello, world!");  
        });

Теперь при нажатии кнопки появится новый текст.
