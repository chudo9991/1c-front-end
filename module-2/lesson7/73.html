7.1.3 Сохранение команд

Чтобы не вводить в терминал длинные команды вручную, их можно сохранять в конфигурационном файле. Для этого в package.json (он появился после npm init --force) нужно использовать словарь scripts.

Это в Brackets!

  1. В папке example открыть package.json
  2. Внести изменения в scripts

        Листинг 7.3 package.json

        {
            "name": "example",
            "version": "1.0.0",
            "main": "script.js",
            "scripts": {
              "watch": "watchify main.js --outfile bundle.js",
              "build": "browserify main.js -- outfile bundle.js"
            },

Теперь в PowerShell доступны сокращенные команды npm run watch и npm run build.
