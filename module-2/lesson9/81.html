Домашка: Фильм и Галерея фильмов

Готовая домашка лежит в папках "1" и "2" в сегодняшнем занятии.

Как запустить у себя:

  • Скопировать всё (файлы и папки). package и bundle не нужны.
  • В Powershell: cd в папку с файлами
  • В Powershell: Set-ExecutionPolicy RemoteSigned -Scope Process
  • В Powershell: npm init --force
  • В Powershell: npm install vue --save
  • В Powershell: npm install vueify babel-core babel-preset-es2015 babel-runtime babel-plugin-transform-runtime --save-dev
  • В Powershell: npm install vue-router --save
  • В Powershell: browserify main.js --outfile bundle.js --transform vueify
  • Запускаем index.html.
