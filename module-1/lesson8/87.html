8.3.3 Кнопки

Это примеры классов кнопок Primary, Warning, Info, Light. С другими классами кнопок Bootstrap можно познакомиться выполняя задание.

Дополнительно. Для изменения размеров кнопок используйте классы btn-sm и btn-lg. Класс btn-block устанавливает ширину кнопки равной ширине родительского элемента. Чтобы сделать кнопку недоступной, используйте класс disabled.


    Листинг

    <button type="button" class="btn btn-primary">Primary</button>
    <button type="button" class="btn btn-warning">Warning</button>
    <button type="button" class="btn btn-info">Info</button>
    <button type="button" class="btn btn-light">Light</button>

Задание для самостоятельной работы!

  • Убедитесь, что у вас подключены Bootstrap и JQuery!
  • Сделайте самостоятельно кнопки secondary, success, danger, dark.
  • Примените классы btn-sm, btn-lg, btn-block, disabled к любым кнопкам по вашему выбору.
  • Покажите результат преподавателю через Light Manager
