8.1 Предпосылки

Для оформления внешнего вида страниц существуют различные библиотеки. Одной из популярных является Bootstrap. Что есть ещё? Можно спросить Гугл про "CSS framework".

Bootstrap предоставляет множество часто используемых визуальных элементов. Некоторые мы рассмотрим на сегодняшнем занятии.

8.1 Подключение

Чтобы начать работу с Bootstrap необходимо подключить файлы для CSS и JavaScript. А также нам понадобиться подключить JQuery.

Вот ссылка на актуальную версию. На сегодняшний день это 4.4.1

Порядок подключения, как и в случае со скриптами ВАЖЕН!

Если 2 каких-либо правила подходят к одному элементу и имеют одинаковую специфичность, то будет применено то, которое объявлено последним.

Значит, библиотечные стили (Bootstrap) должны подключаться РАНЬШЕ, чем написанные вами.

    Листинг 8.1 Подключение (index.html)    
        <!DOCTYPE html>
        <html lang="ru">
        <head>
            <meta charset="uft-8">
            <title>Это заготовки</title>
            <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css">
            <link rel="stylesheet" href="style.css">
        </head>
        <body>
    
            <!-- Здесь будет наш HTML-код! -->

            <script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
            <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/js/bootstrap.min.js"></script>
        </body>
        </html>
