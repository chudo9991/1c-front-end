11.6 Анимация

Добавляем плавное перемещение костяшек в классе field__cell--tile.

        Листинг style.css

        .field__cell--tile {
            background-color: #3D1255;
            color: white;
            font-size: 42px;
            font-weight: bold;
        
            /* центрирование числа на костяшке */
            display: flex;
            justify-content: center;
            align-items: center;

            /*плавное перемещение костяшек*/
            transition: left 0.5s ease, top 0.5s ease;
        }
