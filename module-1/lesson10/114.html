11.2.1 createCellNull

Функция должна возвращать новый элемент div с классами field__cell и field__cell--null.

Обратите внимание, это словарь, т.е. пара ключ-значение!

        Листинг 11.2.1 script.js

        function createCellNull() {
            var cell = document.createElement("div");
            cell.classList.add("field__cell", "field__cell--null");
            return cell;
        }

11.2.2 setCellOffset

Функция должна устанавливать положение переданного ей элемента при помощи стилевых свойств left и top.

Обратите внимание, что элементу cell мы устанавливали свойства x и y:

        cell.y = y;
        cell.x = x;

Они обозначают номер строки и столбца при нумерации с 0. Их можно использовать для расчета положения по формулам:

        left = offset + ( offset + width ) * x
        top = offset + ( offset + height ) * y

Offset - расстояние между ячейками.

        Листинг 11.2.2 script.js

        function setCellOffset(cell) {
            /* + "px" необходимо, так как это значение своиства CSS */
            cell.style.left = (15 + (15 + 81.25) * cell.x) + "px";
            cell.style.top = (15 + (15 + 81.25) * cell.y) + "px";
        }

11.2.3 appendCell

Функция должна добавлять переданный ей элемент к игровому полю.

        Листинг 11.2.3 script.js

        function appendCell(cell) {
            var field = document.getElementById("field");
            field.appendChild(cell);
        }
