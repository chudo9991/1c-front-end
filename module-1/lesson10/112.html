11.1 Игровое поле

Создаем квадратный div с именем (id) field, расположенный в середине страницы. Свойства блока записываем строго при помощи CSS.

        Листинг index.html 

        <!DOCTYPE html>
        <html> 
        <head> 
            <meta charset="utf-8"> 
            <title>Пятнашки</title>
            <link rel="stylesheet" href="style.css">
        </head>
        <body> 
            <div class="field" id="field"> 
            </div> 
            <script src="script.js"></script> 
        </body> 
        </html>
    
        Листинг style.css 

        html, body { 
            height: 100%; 
            margin: 0; 
        } 
        
        body { 
            /* центрирование игрового поля */ 
            display: flex; 
            justify-content: center; 
            align-items: center; 
            font-family: Arial; 
            } 

        .field { 
            width: 400px; 
            height: 400px; 
            background-color: #9775AA; 
            border-radius: 5px; 
            position: relative;
        }
