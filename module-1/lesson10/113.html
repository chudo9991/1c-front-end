11.2 Плитка

Создаем класс field__cell, который будет использоваться как для фоновых, так и для игровых ячеек. Он должен задавать одинаковую ширину и высоту, а так же абсолютное позиционирование игрового поля.

Создаем класс field__cell--null, устанавливающий цвет фоновой ячейки.

Далее напишем функции creatrCellNull, setCellOffset и appendCell, необходимые для работы функции createField.

        Листинг 11.1 script.js

        function createField() { 
            var x, y, cell; 
            for(y = 0; y < 4; ++y) { 
                for(x = 0; x < 4; ++x) { 
                    cell = createCellNull(); 
                    cell.y = y;
                    cell.x = x; 
                    setCellOffset(cell); 
                    appendCell(cell); 
                }
            }
        }
        createField();
    
        Листинг style.css

        .field__cell {
            width: 81.25px; 
            height: 81.25px; 
            position: absolute; 
            border-radius: 3px; 
            }
        
        .field__cell--null {
            background-color: #764B8E; 
        }
