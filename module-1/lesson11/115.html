12.2 Поздравительное сообщение

Внутрь элемента field в index.html добавляем блок div с именем "modal", содержащий текст поздравления.

        Листинг index.html 

        <html>
            <head>
                <meta charset="utf-8"> 
                <title>Пятнашки</title> 
                <link rel="stylesheet" href="style.css"> 
            </head> 
            <body> 
                <div class="field" id="field">
                    <div class="modal" id="modal">   <!-- Добавили -->
                        Победа!
                    </div> 
                </div> 
            <script src="script.js"></script> 
            </body> 
        </html>

    

        Листинг style.css

        #modal { 
            /* размеры равные размеру родителя, 
            то есть игрового поля */ 
            height: 100%; 
            width: 100%; 
            /* наследование радиуса границ */ 
            border-radius: inherit; 
            
            /* невидимость */ 
            visibility: hidden; 
            opacity: 0; 
            transition: opacity 1s ease; 
            
            /*вывод блока на передний план */ 
            position: relative; 
            z-index: 1; 
            
            /* центрирование сообщения */ 
            display: flex; 
            align-items: center;
            justify-content: center; 
            
            /* цвета и размер шрифта */ 
            background: rgba(0, 0, 0, 0.5); 
            color: white; 
            
            font-size: 72px; 
            }
            
        #modal.modal--visible {
            visibility: visible; 
            opacity: 1;
            }

Задание для самостоятельной работы!

  • Переносим код в index.html
  • Добавляем код в style.css
  • Показываем результат в Light Manager
  • При наступлении какого события нужно выводить сообщение о победе?
  • Ответ пишем в чат!
