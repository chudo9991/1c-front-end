9.4 z-index

Ещё раз вернемся к листингу 9.5 и добавим следующее правило:

       Листинг 9.11 .inner.inner--green 

       .inner.inner--green {
           position: relative;
           left: -50px;
           top: 50px;
       }

Зеленый блок оказался к нам 'ближе', чем красный, так как он объявлен позже в коде страницы. Целочисленное свойство z-index позволяет изменить этот порядок. Для его использования требуется свойство position в значении relative, absolute или fixed.

Задание для самостоятельной работы!

  • Добавьте в листинге 9.5 класс .inner.inner--green
  • Добавьте в листинге 9.5 класс .inner.inner--red:active
  • Напишите в чат трансляции что происходит при нажатии на красный блок и почему?
        Листинг 9.8 Класс .inner.inner--red:active

        .inner.inner--red:active {
            position: relative;
            z-index: 1;
        }
