9.3.3 absolute

Работает так же, как absolute, но позиционирует элемент относительно окна. Таким образом реализованы окна в социальных сетях.


То, что вы видите на экране реализовано вот так:

        Листинг 9.9 index.html
        
        <style>
            .body {
                height: 200px;
            }
            .outer {
                border: 1px solid lightblue;
                border-radius: lightgrey;
                height: 600px;
            }
            .inner {
                display: inline-block;
                width: 100px;
                height: 100px;
            }
            .inner--red { background-color: red;}
            .inner--green { background-color: green; }
            .inner:active {
                position: absolute;
                width: auto;
                height: auto;
                left: 0;
                right: 0;
                top: 100px;
                bottom: 100px;
            }
            .inner.inner--red{
                position: fixed;
                bottom: 0;
                right: 0;
            }
            </style>
        
            <div class="outer">
                <div class="inner inner--red">
                </div><div class="inner inner--green">
                </div>
            </div>
