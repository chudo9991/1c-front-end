7.4.5 css

Функция css устанавливает свойства элементам CSS. Значения в словаре рекомендуется делать строками.

        Листинг 7.6 body (index.html)

        <div></div>
        <div></div>
        <script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
        <script>
            $("div").css({
                width:"50px",
                height:"50px",
                margin:"5px",
                backgroundColor:"red"
            });
        </script>
