7.1 Предпосылки

По мере разрастания проектов количество однотипных конструкций, работающих с DOM становится всё большим, из-за этого число ошибок в коде увеличивается. Для решения этой проблемы разработали библиотеку JQuery. А потом и большое количество плагинов для неё.

Современное веб-программирование и создание веб-сайтов уже невозможно представить без использования языка JavaScript. Однако в настоящее время, все чаще используется не "голый" код javascript, а javascript-фреймворки и библиотеки. Одной из таких библиотек, причем наверное самой популярной на сегодняшний день, является jQuery. По некоторым оценкам не менее половины крупнейших сайтов в интернете используют эту библиотеку. Хотя мы и можем назвать jQuery библиотекой, он на самом деле понятие "jQuery" объединяет целую экосистему библиотек, построенный вокруг базовой библиотеки: это и библиотека jquery.ui, предназначенная для создания визуальных интерфейсов, это и jqyery.mobile, используемая при разработке мобильных сайтов и др.

7.2 Как подключить JQuery?

        Листинг 7.2

        <body>
            ...ваш HTML-код...
            <script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
        </body>

7.3 Синтаксис

Большинство команд в JQuery записывается так:

            $(селектор).действие();

Эта команда примени действие в элементам, выбранным при помощи селектора. $ - специальная функция, которая возвращает массив соответствующих объектов.
Несколько действий можно связать вот в такую цепочку.

            $(селектор).действие-1().действие-2();
