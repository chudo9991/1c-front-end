7.4 Функциональность

7.4.6 width и height

Функции width и height возвращают или устанавливают длину или ширину элементов соответственно.

        Листинг 7.8 body (index.html)

        <div class="block"></div>
        <script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
        <script>
            $("*.block").width(100).height(50);
            alert($(".block").width())
        </script>

Подключаем style.css:

        <head>
        ...
        <link rel="stylesheet" href="style.css">
        </head>

В style.css прописываем:

        Листинг 7.7 style.css

        .block {
            background-color: red;
        }
