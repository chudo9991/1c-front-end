7.4 Функциональность

7.4.7 addClass, removeClass и toggleClass

Функции addClass, removeClass и toggleClass добавляют, удаляют и переключают классы элементов соответственно.

        Листинг 7.8 body (index.html)

        <div class="block"></div>
        <button type="button" id="switch">ВКЛ/ВЫКЛ</button>
        <script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
        <script>
            $("#switch").on("click", function(){
                $(".block").toggleClass("block--lit");
            });
        </script>

Подключаем style.css:

        <head>
        ...
        <link rel="stylesheet" href="style.css">
        </head>

В style.css прописываем:

        Листинг 7.9 style.css

        .block {
            width: 50px;
            height: 50px;
            margin: 10px;
            background-color: black;
        }
        
        .block--lit {
            background-color: yellow;
        }
