В этом примере мы уберем проверку на нажание клавиши enter при вводе нового дела.

Список дел (модифицированный)

  1. {{ item }}

Мы заменили вот такой код в HTML:

        <input type="text" v-model="thing" v-on:keydown="addItem($event)">

На вот такой:

        <input type="text" v-model="thing" v-on:keydown.enter="addItem">

Или вот такой:

        <input type="text" v-model="thing" v-on:keydown.13="addItem">

И изменили в 64mod.js вот этот код:

        addItem: function (event) {
            if (event.which==13) { 
                this.list.push(this.thing); 
                this.thing="";
            }
        },

На вот такой. При этом мы убрали не нужную более проверку на нажатую клавишу, перенеся её в HTML.

        addItem: function (event) {
            this.list.push(this.thing); 
            this.thing="";
            }
        },
Для этих клавиш можно использовать псевдонимы вместо кодов:
  • латинские буквы от a до z
  • enter
  • tab
  • delete
  • esc
  • space
  • up, down, left, right - стрелочки
