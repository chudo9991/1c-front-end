Начнем с простого примера. Написанная нами программа при нажатии на кнопку "Удвоить" число будет увеличиваться в 2 раза.

Происходить это будет за счет обработчика событий.

{{number}}

Обработчик событий - ждет пока не случится то или иное событие. Нажатие клавиши, клик на определенном элементе и т.д. Как только событие случилось - обрабатывает его!

Обрабатывают события - методы. Они же - процедуры и функции. Но в ООП их принято называть методами, так как они являются частью объекта.

Их объявляют в словаре methods. У нас листинг 6.2 "спрятан" в файл 62.js.

            Листинг 6.2

            let vm = new Vue ({
                el: "#app",
                data: {
                    number : 1
                    },
                methods : {
                    double: function(event) {
                        console.log(event);
                        this.number *=2;
                    }
                }
            });

При этом сам vm и есть объект!

В тексте этой страницы есть вот такой код, заставляющий всё работать.

            <div id="app">
                <button type="button" v-on:click="double">Удвоить</button>
                <p>{{number}}</p>
            </div>
