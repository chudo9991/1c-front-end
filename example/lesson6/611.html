Добавим немного красоты! С помощью тега transition можно задавать анимации появления и исчезновения элементов.

Скрываем элементы (модифицированный):

Now You See Me!

Код в HTML:

        <transition name="fade">
            <h1 v-show="visible">Now You See Me!</h1>
        </transition>

Добавили подключение CSS в раздел HEAD:

        <link rel="stylesheet" href="611.css">

В файле стилей 611.css прописали следующее:

        .fade-enter-active, .fade-leave-active {
            transition: opacity 1s ease;
        }
        
        .fade-enter, .fade-leave-to {
            opacity: 0;
        }

Описание необходимых классов:

  • *-enter-active - описывает свойства объекта во время появления
  • *-leave-active - описывает свойства объекта во время исчезновения
  • *-enter - описывает свойства объекта в начале появления
  • *-leave-to - описывает свойства объекта в конце исчезновения
