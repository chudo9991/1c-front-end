7.2.3 Структура

Добавим файл header.vue с кодом новой компоненты.

Теперь мы можем использовать её в любых шаблонах. Зададим в главной компоненте тег-соответствие при помощи словаря components.

Скомпилируем и убедимся в наличии синей надписи.

Обратите внимание c-tag заменился заголовком h1.

Команда для PowerShell: 'browserify main.js --outfile bundle.js --transform vueify'.

        Листинг 7.11 header.vue

        <style scoped>
            h1{
                color: blue;
            }
        </style>
        
        <template>
            <h1>{{msg}}</h1>
        </template>
        
        <script>
            module.exports={
                data: function() {
                    return {
                        msg: "Hello, component!"
                    };
                }
            };
        </script>
    
        Листинг 7.12 app.vue
    
        <template>
            <div>
                <c-tag></c-tag>
            </div>
        </template>
        <script>
            let Header=require("./header.vue");
            
            module.exports={
                components: {cTag:Header}
            };
        </script>
